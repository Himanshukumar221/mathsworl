📚 MATHS WORLD MUNGER
MATHS WORLD MUNGER ek dedicated educational platform hai jise Munger, Bihar ke students ke liye design kiya gaya hai. Iska maqsad 10th aur 12th ke students ko behtar coaching facilities provide karna aur digital management (attendance, fees, results) ko asaan banana hai.

🚀 Features
Admin Dashboard: Teachers ke liye attendance aur result upload karne ki facility.

Student Portal: Students apne fee status aur exam results check kar sakte hain.

Modern UI/UX: Glassmorphism aur 3D responsive design jo mobile aur desktop dono par smooth chalta hai.

Easy Navigation: Simple aur clean interface.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Glassmorphism & 3D Effects), JavaScript.

Backend: Java Servlets, Jakarta EE.

Database: MySQL (Relational Database).

Server: Apache Tomcat.
